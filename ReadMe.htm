Beispiel für Befehlszeilenparameter

Beispieldateien:

In diesem Beispiel wird demonstriert, wie auf die Befehlszeile zugegriffen wird und welche beiden Methoden für den Zugriff auf das Befehlszeilenparameter-Array verfügbar sind.

So öffnen Sie die Beispieldatei im Projektmappen-Explorer

  1. Klicken Sie auf Beispiel downloaden.

    Das Meldungsfeld Dateidownload wird angezeigt.

  2. Klicken Sie auf Öffnen. Klicken Sie anschließend im Fenster des ZIP-Ordners in der linken Spalte auf Alle Dateien extrahieren.

    Der Extrahier-Assistent wird geöffnet.

  3. Klicken Sie auf Weiter. Sie können das Verzeichnis ändern, in das die Dateien extrahiert werden. Klicken Sie anschließend erneut auf Weiter.

    Stellen Sie sicher, dass das Kontrollkästchen Extrahierte Dateien anzeigen aktiviert ist. Klicken Sie auf Fertig stellen.

  4. Doppelklicken Sie auf die SLN-Datei des Beispiels.

    Die Beispielprojektmappe wird im Projektmappen-Explorer angezeigt. Sie erhalten unter Umständen die Sicherheitswarnung, dass der Speicherort der Projektmappe nicht vertrauenswürdig ist. Klicken Sie auf OK, um fortzufahren.

So erstellen Sie die Beispiele für Befehlszeilenparameter in Visual Studio und führen sie aus

  1. Klicken Sie im Projektmappen-Explorer mit der rechten Maustaste auf das Projekt CmdLine1, und wählen Sie Als Startprojekt festlegen.

  2. Klicken Sie im Projektmappen-Explorer mit der rechten Maustaste auf das Projekt, und klicken Sie auf Eigenschaften.

  3. Öffnen Sie den Ordner Konfigurationseigenschaften, und klicken Sie auf Debuggen.

  4. Geben Sie in der Eigenschaft Befehlszeilenargumente die Befehlszeilenparameter ein, und klicken Sie auf OK. (Ein Beispiel finden Sie im Lernprogramm.)

  5. Klicken Sie im Menü Debuggen auf Starten ohne Debuggen.

  6. Wiederholen Sie die vorangehenden Schritte für CmdLine2.

So erstellen Sie die Beispiele für Befehlszeilenparameter über die Befehlszeile und führen sie aus

  1. Wechseln Sie mithilfe des Befehls Change Directory in das Verzeichnis CmdLine1.

  2. Geben Sie Folgendes ein:

     Code kopieren
    csc cmdline1.cs
cmdline1 A B C

  3. Wechseln Sie mithilfe des Befehls Change Directory in das Verzeichnis CmdLine2.

  4. Geben Sie Folgendes ein:

     Code kopieren
    csc cmdline2.cs
cmdline2 John Paul Mary
